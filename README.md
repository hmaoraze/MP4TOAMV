## 🚀 快速开始

### 安装 FFmpeg

1. 访问 FFmpeg 官网：
   👉 [https://ffmpeg.org/download.html](https://ffmpeg.org/download.html)
2. 在 **Windows** 区域选择 **Gyan.dev** 版本（推荐）
3. 点击 **Release builds**，下载：

   ```
   ffmpeg-release-essentials.zip
   ```

   （约 70–80 MB）
4. 解压到任意目录，例如：

   ```
   C:\ffmpeg
   ```

   或

   ```
   D:\Tools\ffmpeg
   ```
5. 记住解压后的路径

---

### 配置环境变量

> 以下步骤适用于 Windows 10 / 11

1. 按 **Win + R**，输入：

   ```
   sysdm.cpl
   ```

   回车
2. 打开 **高级** → **环境变量**
3. 在「系统变量」或「用户变量」中找到 **Path** → **编辑**
4. 点击 **新建**，填入 FFmpeg 的 `bin` 目录，例如：

   ```
   C:\ffmpeg\bin
   ```
5. 一路点击 **确定** 保存

---

### 验证安装

打开 **命令提示符（CMD）**，输入：

```bash
ffmpeg -version
```

* ✅ **成功**：显示 FFmpeg 版本信息
* ❌ **失败**：提示“不是内部或外部命令” → 请重新检查环境变量配置

## 🛠 使用说明

1. **运行工具**
   双击：

   ```
   MP4TOAMV_1.0_amd64.exe
   ```
2. **选择文件**
   点击「选择视频文件」，可批量勾选多个 MP4 文件
3. **开始转换**
   点击「开始转换」，通过进度条查看实时状态
4. **查看结果**
   转换后的 `.amv` 文件将保存在**源文件所在目录**

> 💡 **建议**：单次转换不超过 **10 个文件**，以避免系统资源占用过高

## ❓ 常见问题

### Q1：工具提示“找不到 FFmpeg”？

**A：** 请确认：

* FFmpeg 已正确解压
* `bin` 目录已加入 `Path` 环境变量
* 重启 CMD 或电脑后再试

---

### Q2：转换速度较慢？

**A：** 转换速度与以下因素有关：

* 视频文件大小
* CPU / 磁盘性能

默认参数已针对 AMV 格式进行优化，一般无需额外调整。

---

### Q3：转换失败怎么办？

**A：** 请依次排查：

1. 源视频文件是否损坏
2. 磁盘剩余空间是否充足
3. 文件是否被其他程序占用

如仍失败，建议升级 FFmpeg 至最新版本。

---

### Q4：AMV 格式有什么限制？

**A：** 本工具使用的 AMV 参数如下：

* 分辨率：**160 × 120**
* 帧率：**15 FPS**
* 音频：**单声道 22.05 kHz**

适用于老式 MP4 / AMV 便携播放器。

---

### Q5：如何卸载 FFmpeg？

**A：**

1. 删除 FFmpeg 解压目录
2. 从环境变量 `Path` 中移除对应路径
3. 重启电脑
